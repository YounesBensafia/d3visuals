# Data Visualization Projects

A collection of interactive data visualizations demonstrating various chart types and statistical analysis techniques.

## Visualizations

- **scatter-plot** - Social media engagement analysis with interactive scatter plots
- **bar-chart** - Bar chart visualizations with sorting and transitions
- **us-cities-population** - US cities population distribution
- **sorting-animation** - Animated sorting algorithm visualization
- **stacked-bar-chart** - Multi-category stacked bar charts (Titanic survival data)
- **histogram** - Age distribution histograms with binning
- **kernel-density-estimation** - Kernel density estimation plots for distribution analysis
- **boxplot** - Statistical boxplot with outlier detection
- **bubble-chart** - Multi-dimensional bubble chart visualization
- **treemap** - Hierarchical treemap for categorical data
- **treemap-hierarchical** - Multi-level hierarchical treemap
- **pca-analysis** - Principal Component Analysis with biplot visualization
- **correlogram** - Correlation matrix heatmap

## Getting Started

```bash
# Clone the repository
git clone https://github.com/YounesBensafia/data-viz-labs.git
cd data-viz-labs

# Start a local server
python -m http.server 8000
# or
npx serve

# Open in browser
# http://localhost:8000/visualizations/scatter-plot/
```
